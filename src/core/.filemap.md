# Core Module (`frontend/src/core`)

Three.js scene, camera, and renderer initialization.

## Files

| File | Purpose |
|------|---------|
| `index.ts` | Core class: creates and configures scene, camera, renderer |

## Core Class Components

### Camera

- Type: `PerspectiveCamera`
- FOV: 50
- Near: 0.01, Far: 500
- Initial Position: (10, 5, 10)
- Initial LookAt: (0, 0, 0) - center of ground plane

### Scene

- Background: Sky blue (`0x87ceeb`)
- Fog: Disabled for clearer visibility
- Lighting:
  - Sun light 1: PointLight at (500, 500, 500), intensity 0.5
  - Sun light 2: PointLight at (-500, 500, -500), intensity 0.2
  - Ambient: AmbientLight (`0x404040`)

### Renderer

- Type: `WebGLRenderer`
- Size: Window dimensions (responsive)
- Target: Appended to `document.body`

## Resize Handling

Both camera and renderer update on window resize:
- Camera: Updates aspect ratio and projection matrix
- Renderer: Updates size to window dimensions

## Related Files

- Main (instantiation): `main.ts`
- Terrain (scene usage): `terrain/index.ts`
- Control (camera usage): `control/index.ts`

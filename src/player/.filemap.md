# Player Module (`frontend/src/player`)

Player state, movement modes, and speed values.

## Files

| File | Purpose |
|------|---------|
| `index.ts` | Player class, Mode enum, Speed enum |

## Mode Enum

```typescript
enum Mode {
  walking = 'walking',
  sprinting = 'sprinting',
  flying = 'flying',
  sprintFlying = 'sprintFlying',
  sneaking = 'sneaking'
}
```

Note: MVP uses `flying` mode exclusively.

## Speed Enum

```typescript
enum Speed {
  walking = 5.612,
  sprinting = 5.612,
  flying = 21.78,
  sprintFlying = 21.78,
  sneaking = 2.55
}
```

## Player Class

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `mode` | Mode | `Mode.flying` | Current movement mode |
| `speed` | number | `Speed.flying` | Current speed (units/second) |
| `falling` | number | 38.4 | Gravity acceleration (unused in flying) |
| `jump` | number | 1.2522 | Jump velocity (unused in flying) |
| `body.height` | number | 1.8 | Player height for collision |
| `body.width` | number | 0.5 | Player width for collision |

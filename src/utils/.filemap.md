# Utils Module (`frontend/src/utils`)

Shared utility functions.

## Files

| File | Purpose |
|------|---------|
| `index.ts` | DOM helpers, mobile detection, color conversion |

## Exports

### `htmlToDom(html: string)`

Converts HTML string to DOM and appends to document body.
Used by joystick module for loading HTML templates.

### `isMobile: boolean`

Constant that checks user agent for mobile devices:
Android, webOS, iPhone, iPad, iPod, BlackBerry

### `blockTypeToHex(type: BlockType): string`

Converts BlockType enum value to hex color string.

| BlockType | Color |
|-----------|-------|
| red | `#FF0000` |
| orange | `#FFA500` |
| yellow | `#FFFF00` |
| green | `#00FF00` |
| blue | `#0000FF` |
| violet | `#8B00FF` |
| brown | `#8B4513` |
| white | `#FFFFFF` |
| gray | `#808080` |
| black | `#000000` |
| (fallback) | `#808080` (gray) |

## Usage

- **Control** - Uses `blockTypeToHex` when creating new blocks
- **UI** - Uses `isMobile` for pointer lock handling, `blockTypeToHex` for backward compatibility
- **Bag** - Uses `isMobile` to skip initialization on mobile
